# D3.js Dashboard

A React dashboard that visualizes COVID-19 data using various D3.js chart types.

## Data Setup

1. Download the COVID-19 dataset CSV from [Our World in Data](https://covid.ourworldindata.org/data/owid-covid-data.csv)
2. Create a `public/data` directory in your project root if it doesn't exist:
   ```
   mkdir -p public/data
   ```
3. Copy the downloaded CSV file to the data directory and rename it:
   ```
   cp path/to/downloaded/file.csv public/data/covid-data.csv
   ```
